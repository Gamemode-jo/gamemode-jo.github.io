O que é este Hub?

Este site é um centro de comando digital

A ideia surgiu da necessidade de organizar vários projetos e ideias diferentes em um só lugar, de forma prática e acessível.

Aqui, cada card no menu principal leva a um experimento, ferramenta ou site independente, todos hospedados pelo GitHub já que eu não tinha dinheiro para pagar por algo.
